Visual Studio Code configuration
================================

Install extensions
------------------

```bash
$ ./install-extensions
```

Install settings
----------------

```bash
$ ./install-settings
```

Install keybindings
-------------------

```bash
$ ./install-keybindings
```

Update extension list
---------------------

```bash
$ ./update-extension-list
```

Update settings file
--------------------

```bash
$ ./update-settings-file
```

Update keybindings file
-----------------------

```bash
$ ./update-keybindings-file
```
